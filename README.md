# EmployeeManagementSystem
A full-stack web application to manage employee records with complete CRUD functionality (Create, Read, Update, Delete). Implemented RESTful APIs using Spring Boot and integrated them with an Angular frontend for seamless user interaction.

# Employee Management System

Full-stack Employee Management System built using Angular, Spring Boot, and MySQL.

## Features
- Add Employee
- Update Employee
- Delete Employee
- View Employee Details
- List All Employees

## Tech Stack
- Angular
- Spring Boot
- MySQL
- REST API

## Run Frontend
cd frontend
npm install
ng serve

## Run Backend
cd backend
mvn spring-boot:run
